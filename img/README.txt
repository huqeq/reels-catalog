Проект: Каталог блогеров для рекламы (сторис и рилс)
===================================================

СТРУКТУРА ПРОЕКТА
-----------------
/project
   index.html      - главная страница с описанием и кнопкой перехода
   bloggers.html   - страница с каталогом блогеров
   style.css       - общий файл стилей (неоновый/стеклянный дизайн)
   script.js       - общий JavaScript (анимации и генерация карточек)
   bloggers.js     - массив данных с блогерами
   /img/           - папка с фотографиями блогеров
   README.txt      - этот файл


1. КАК ОТКРЫТЬ САЙТ
-------------------
1) Скопируйте всю структуру проекта в папку (например, "project").
2) Убедитесь, что файлы находятся именно так:

   project/index.html
   project/bloggers.html
   project/style.css
   project/script.js
   project/bloggers.js
   project/img/...

3) Откройте файл "index.html" двойным кликом через браузер (Chrome, Edge, Safari и т.д.).
4) На главной странице нажмите кнопку "Перейти к блогерам" чтобы открыть каталог.
   При желании можно сразу открыть "bloggers.html" напрямую.


2. КАК ДОБАВИТЬ НОВОГО БЛОГЕРА
------------------------------
Все блогеры хранятся в файле "bloggers.js" в массиве "window.bloggers".

Пример структуры объекта:

{
  name: "Имя блогера",
  photo: "img/имяфайла.jpg",
  description: "Краткое описание",
  reels: "от 5000₽",
  stories: "от 3000₽",
  category: "Ниша / тематика"
}

Чтобы добавить нового блогера:

1) Откройте "bloggers.js" в любом текстовом редакторе.
2) В массиве "window.bloggers = [ ... ]" добавьте новый объект через запятую.

Например:

window.bloggers = [
  {
    name: "Алина Маркова",
    photo: "img/blogger1.jpg",
    description: "Описание...",
    reels: "от 15 000₽",
    stories: "от 8 000₽",
    category: "Lifestyle / Beauty"
  },
  {
    name: "Новый Блогер",
    photo: "img/new_blogger.jpg",
    description: "Краткое описание нового блогера.",
    reels: "от 12 000₽",
    stories: "от 6 000₽",
    category: "Ниша / тематика"
  }
];

3) Сохраните файл.
4) Положите файл фотографии (new_blogger.jpg) в папку "img".
5) Обновите страницу "bloggers.html" в браузере – новый блогер появится автоматически.


3. КУДА ЗАГРУЖАТЬ ФОТО
----------------------
Все фото должны находиться в папке:

   /project/img/

В поле "photo" указывайте путь относительно корня проекта, например:

   "img/blogger1.jpg"
   "img/arina_travel.png"

Рекомендации по фото:
- Квадратный формат (1:1) выглядит лучше всего.
- Размер 600x600 или выше (но без огромных файлов по 10 МБ).
- Формат: JPG или PNG.


4. КАК ИЗМЕНИТЬ ЦВЕТА И ЭФФЕКТЫ
-------------------------------
Все основные цвета и эффекты вынесены в CSS-переменные в начале файла "style.css":

:root {
  --bg-color: #050712;
  --bg-elevated: rgba(17, 24, 39, 0.75);
  --border-glass: rgba(255, 255, 255, 0.1);
  --text-main: #f9fafb;
  --text-muted: #9ca3af;
  --accent1: #4f46e5;
  --accent2: #22d3ee;
  --accent3: #ec4899;
  --shadow-neon: 0 0 25px rgba(79, 70, 229, 0.6);
  ...
}

Чтобы изменить стиль:

1) Смена акцентных цветов:
   - Правьте переменные:
     --accent1 (основной неоновый синий/фиолетовый)
     --accent2 (голубой/бирюзовый)
     --accent3 (розово-пурпурный)
   - Этим вы поменяете оттенок градиентов и подсветки.

2) Смена фона:
   - Можно изменить фон в html/body (градиент) на более светлый или более тёмный.

3) Мощность неонового свечения:
   - Регулируется переменной:
     --shadow-neon: 0 0 25px rgba(...);

4) Скорость анимаций:
   - Есть две переменные:
     --transition-fast
     --transition-normal
   - Увеличьте значения (0.35s -> 0.5s), если нужно более плавное поведение.

5) Изменение радиусов:
   - Радиусы скруглений:
     --radius-lg
     --radius-md
   - Можно сделать карточки более "квадратными" или наоборот более круглыми.

После изменений сохраните "style.css" и обновите страницу.


5. КАК УВЕЛИЧИТЬ КОЛИЧЕСТВО БЛОГЕРОВ
------------------------------------
Структура полностью масштабируется: просто добавляйте новые объекты в массив "window.bloggers" в файле "bloggers.js".

Никаких дополнительных изменений в HTML/JS делать не нужно:
- сетка автоматически подстроится под 3xN (на десктопе),
- на планшетах/телефонах сработает адаптивная верстка (2 в ряд, затем 1).


6. ГДЕ ПРАВИТЬ ТЕКСТЫ НА ГЛАВНОЙ
--------------------------------
Все тексты главной страницы находятся в файле "index.html":

- Приветствие, описание, список "что вы найдёте" – в блоках:
  .hero-content, .hero-subtitle, .hero-list, .hero-preview и т.д.

Можете смело править только текст внутри тегов:
<p>...</p>, <li>...</li>, <h1>...</h1>, etc.

Дизайн (классы, структура) лучше не ломать, чтобы не развалить адаптивность и анимации.
